# OpenCV.js for PlanAI Field Panorama

Panorama stitching loads OpenCV.js on first use.

## Offline (recommended for field APK)

1. Download `opencv.js` from https://docs.opencv.org/4.9.0/opencv.js
2. Save as `libs/opencv/opencv.js`

## Online fallback

If the local file is missing, the app loads OpenCV from `docs.opencv.org` (requires network on first panorama).
